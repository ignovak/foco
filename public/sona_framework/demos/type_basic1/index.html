Type demo

On this page we will use some cool features from type module.

To work with type module we need to set @rhythm ans @base_fs variables.

  @rhythm: 20px;
  @base_fs: 16px;

Next step is to properly set body's font-size. Also apply default stylesheet to headers.

  body {
    .type_body();
    .type_headers();      
  }

Now body has 16/20 font. And headers are from 48px to 16px

In article bellow we redefine base font size to 18/20 font. And set hesers sizes from 38 to 18

  article {
    .fs(18px);
    .line(18px);
    .type_headers(38px);
  }

But also we can change styles for headers inside `section.headers` from 72 to 24 

section.headers{
  .fs(24px, 18px); // specify font size from article
  .line(24px);
  .type_headers(72px);
}
